# Assets

Asset categories:

- `brand/logo/`: primary and secondary logo files.
- `brand/icons/`: integration, channel, and brand icons.
- `ui/`: UI-only images.
- `illustrations/`: editorial/demo illustrations.
- `demo/`: sample/demo-only images.
- `product-images/`: product, food, retail, or menu images.

Do not mix customer uploads with static UI assets. Do not delete assets without checking references and updating `docs/ASSET_INVENTORY.md`.
